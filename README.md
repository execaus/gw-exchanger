# gw-exchanger

## Описание проекта

`gw-exchanger` — это сервис хранения курсов валют.

## Инструкция по запуску

### Сборка

```bash
go build -o gw-exchanger ./cmd/main.go
```

### Запуск

1. Настройте файл конфигурации `config.yaml` или задайте переменные окружения (см. ниже).
2. Запустите сервис:

```bash
./gw-exchanger -c ./config.env
```

## Поддерживаемые валюты

Сервис поддерживает любые валюты через расширение списка в базе данных.

### Окружение

Параметры подключения к базе данных, порты и другие настройки задаются через файл `config.env` или переменные окружения:

| Параметр              | Описание                         | Переменная окружения |
|-----------------------|---------------------------------|----------------|
| `server.port`         | Порт для запуска сервера        | `SERVER_PORT`  |
| `database.host`       | Хост базы данных                | `DATABASE_HOST` |
| `database.port`       | Порт базы данных                | `DATABASE_PORT` |
| `database.user`       | Пользователь базы данных        | `DATABASE_USER` |
| `database.password`   | Пароль базы данных              | `DATABASE_PASSWORD` |
| `database.name`       | Имя базы данных                 | `DATABASE_NAME` |
